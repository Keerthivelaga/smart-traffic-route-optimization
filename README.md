# Run

1. Open Android Studio.
2. Open the `android_app` folder.
3. Wait for Gradle Sync to finish.
4. Start an emulator or connect a device with internet access.
5. Run the `app` configuration.
6. Do not run backend or ML locally; the app is already configured to use deployed cloud services.
7. Login is optional; continue as guest to browse map/routing. Login is required for live GPS cloud sync, incident reporting, personalized profile, and leaderboard scoring.
8. Weather-aware routing uses Open-Meteo (`https://api.open-meteo.com/v1/forecast`) and shows route weather on selected routes.

9. For physical devices, keep Location ON (High accuracy) and grant Fine/Coarse location permission when prompted.
